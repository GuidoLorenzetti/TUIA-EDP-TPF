TUIA-EDP

Trabajo Práctico Final

El repositorio consiste en un conjunto de scripts para el ánalisis de un texto en particular. El texto a elegir, debe ser cargado en la carpeta Textos, el código se encargará de mostrarselo al usuario para su posterior ánalisis a partir de los scripts.

Los comandos a ejecutarse en la consola son:

docker build -t tp-edp .
docker run -it tp-edp 

Aclaración: El texto elegido debe estar previamente cargado en la carpeta Textos.
